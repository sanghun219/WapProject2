■555 UI Sound FX
----------------------------------------------------------------------------------------------

この度は効果音アセット｢555 UI Sound FX｣をご購入いただき、ありがとうございます。

｢555 UI Sound FX｣は5つのフォルダで構成されています。


・｢1.select｣	：セレクト、選択音に向いている音
・｢2.OK」	；決定音に向いている音
・｢3.cancel｣	：キャンセル音に向いている音
・｢4.common｣	：幅広い用途に使用可能な音
・｢5.unstable｣	：ピッチが不安定な変わった音


収録データ形式はwavファイル(44.1kHz 16bit)、oggファイルとなっています。


ファイル名に｢_a｣などのアルファベットが記載されているものは同じ音色で制作した効果音で、
それに続く番号(_01など)とキーを合わせてあります。

例）｢ok_a_01｣と｢cancel_a_01｣は同じ音色、同じキーで制作されています。

同じアルファベットと番号のファイルを使用し組み合わせると統一感が出しやすいかと思います。
